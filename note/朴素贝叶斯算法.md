# 朴素贝叶斯算法
概率问题：
#如果所求属于类别-1：概率公式为
* P(3\*M)\*P(-1|3\*M)=P(-1)\*P(3\*M|-1)
  P(-1|3\*M)=P(-1)\*P(3\*M|-1)/P(3\*M)
  P(-1|3\*M)=P(-1)\*P(3|-1)\*P(M|-1)/P(3)*P(M)
#如果所求属于类别1：
* 概率公式为P(3\*M)\*P(1|3\*M)=P(1)\*P(3\*M|1)
  P(1|3\*M)=P(1)\*P(3\*M|1)/P(3\*M)
  P(1|3\*M)=P(1)\*P(3|1)\*P(M|1)/P(3)*P(M)
